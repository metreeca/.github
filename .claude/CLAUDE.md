# Project Overview

This is the [metreeca/.github](https://github.com/metreeca/.github) repository, providing organisation-level default
community health files for all repositories under the [metreeca](https://github.com/metreeca) GitHub organisation.

Files placed here serve as defaults for any metreeca repository that does not define its own version.
